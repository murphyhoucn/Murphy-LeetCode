# 0088 Merge Sorted Array

- `Simple`
- `C++`, `递归`, `动态规划DP`, 

## 问题概述


## 解题思路



``` C++
class Solution
{
public:
	void merge(std::vector<int>& nums1, int m, std::vector<int>& nums2, int n)
	{
		int i = m - 1;
		int j = n - 1;
		int k = m + n - 1;

		while (j >= 0)
		{
			if (i >= 0 && nums1[i] > nums2[j]) { nums1[k--] = nums1[i--]; }
			else { nums1[k--] = nums2[j--]; }
		}
	}
};
```
